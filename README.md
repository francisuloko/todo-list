## Clone Repo

To get a local copy up and running follow these simple steps:

   - `Open terminal Ctrl + Alt + T`

   - `git clone git@github.com:francisuloko/to-do-list.git`

   - `cd to-do-list`

   - `npm install`

   - `npm start`

   - `visit localhost:8080`
